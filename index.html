A Few Accomplishments

Some of the project I have worked on so far.

Designed and built a LiveOps system

When working as a consultant for Netlight, one of the projects I've worked on was to build a LiveOps system for a well known game. The purpose was to be able to deliver a backend driven user experience, that should be able to develop and adapt depending player behaviour. My role in this was to work closely with game designers and design and implement a system that would fufil their needs.

Designed and implemented cross platform play

Another project I've done when working with Netlight was to design and implement a Cross Platform play system that would allow PC, Switch, PS4 and Xbox players to play together in the same session and this was done for a well known AAA game.

Designed and implemented a vision system for Autonomous bus

Scania is one of the biggest truck manufacturers in Sweden, developing and producing trucks that are considered to be one of the best in the business considering quality and modularization. One of the pre development projects that Scania worked on was their autonomous bus that would populate a bus line in the northern part of Stockholm and one of the big challenges in that project was to make it aware of its surroundings in a noisy environment and the type of sensor we worked with was an array of cameras covering all angles around the bus, which made it challenging due to the high amount of data that needed to be processed by complex image processing machine learning algorithms in a essential time critical safety system. This project was one part of the lucrative new area of autonomous vehicles, to gain an edge over their competitors and position them self as a big contender in that segment.

Published article at ITSC 2016, Probibalistic collision estimation system for autonomous vehicles

I took a leading role in the development and the architecture design of this system as well as key parts of the estimation engine in the system. The goal was to create both a simulation environment that could be used to test and validate the system as well as designing the system that would do the actual calculations and control of the car. The simulation was done in ROS and the system was developed in C++. The results were very successful which lead to that I also wrote a paper on the project (Probabilistic collision estimation system for autonomous vehicles - ITSC 2016).

Built test software for a electrical enginges hydraulic break

Hartvik Engineering AB is a small company developing electronic designs and embedded software for industrial purposes and their niche is low powered systems, radio modules and control systems. One of their customers had the need to do automated testing of high powered electric motors. So they created a hydraulic brake bench for this, but did not have a good way of controlling it. So they needed help with a system where they could connect a motor to the break bench and then from a computer start a “test cycle” where they defined how fast the motor was going to run and how much resistance it should have.
